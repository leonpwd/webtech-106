# Web Technologies Labs

This repository contains lab exercises for the Web Technologies course.

## Lab 1: Introduction to JavaScript
This lab covers the basics of JavaScript, including:
- Variables and data types
- Operators and expressions
- Control structures (if, for, while)
- Functions
- Basic DOM manipulation

## Lab 2: Basic HTTP Server
This lab introduces the concept of HTTP servers. You will:
- Set up a simple HTTP server using Node.js
- Handle basic HTTP requests and responses
- Serve static files
- Understand request/response lifecycle

---
Each lab folder contains its own code and instructions.
