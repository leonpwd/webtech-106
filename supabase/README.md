Supabase integration scaffold

This folder contains an example Supabase client helper and an env template.

Steps to use:
1. Create a Supabase project and get `SUPABASE_URL` and `SUPABASE_ANON_KEY`.
2. Copy `.env.example` to `.env.local` and set values.
3. Use `supabase/client.ts` to connect from server-side code.

Do not commit real keys to the repository.
